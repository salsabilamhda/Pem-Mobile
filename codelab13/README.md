# Praktikum 3
## Soal 5 

- Jelaskan maksud kode lebih safe dan maintainable!

  Jawab :

  Penggunaan *konstanta* (const keyId, const keyName) alih-alih string literal ('id', 'pizzaName') membuat kode lebih *aman (*safe)* karena menciptakan satu sumber kebenaran (*Single Source of Truth), sehingga meminimalkan risiko kesalahan pengetikan (typo) kunci JSON yang sulit dideteksi (runtime error) dan memungkinkan IDE mendeteksi kesalahan saat kompilasi (compile-time error); kode juga menjadi lebih mudah *dipelihara (*maintainable)* karena jika struktur JSON eksternal berubah, pengembang hanya perlu memodifikasi nama konstanta di **satu tempat* saja, memastikan konsistensi di seluruh fungsi seperti fromJson dan toJson.
